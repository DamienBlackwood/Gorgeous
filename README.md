# gorgeous

my obsidian theme. 

serif headings, purple accent, dark as possible. I'm still learning how to implement css into obsidian's theming system so it's a heavy WIP.

 I used [@kepano's Minimal](https://github.com/kepano/obsidian-minimal) *a lot* as a reference for how to do things properly, so thank you.




## what it looks like

- georgia serif headings, clean sans body text
- a really dark dark mode
- purple accents throughout, links, checkboxes, active items
- blockquotes are just a quiet bar on the left, no box
- h6 is uppercase and small, kind of like a label

## installing

**not yet on community themes**

**manual method:** clone this repo into `.obsidian/themes/` in your vault, then pick it in Settings → Appearance
